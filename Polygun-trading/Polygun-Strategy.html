Setup Sequence
1
Start bot & deposit
Open PolyGun → deposit USDC on Polygon. Min $50 recommended to run all 4 wallets simultaneously.
2
Add The Steady first
Copy Trading → Add Copy Trade → paste Wallet 3 address → Fixed $15 → price range 0.30–0.90 → slippage 5% → max spend $75.
3
Add The Limit King
Percentage mode 7% → enable limit order copying → set price offset -1% → range 0.25–0.80 → slippage 3%.
4
Add The Whale
Fixed $5 per trade (not more) → set daily cap $50 → range 0.35–0.85. High frequency means fees stack fast without a cap.
5
Add The Contrarian (optional)
Fixed $3 max → range 0.05–0.40 (longshots only) → slippage 8% → daily cap $20. Treat as lottery tickets, not core strategy.
6
Daily check-in
Review PnL per wallet each day. If any wallet goes on a 3+ loss streak, pause it. Re-evaluate weekly via polymarketanalytics.com/traders.
⚠ Risk Reminder Past win rates don't guarantee future results. Only use capital you can afford to lose. Start with smaller fixed amounts for the first 1–2 weeks before scaling up. The Contrarian wallet is highly speculative — treat it as a small side bet only.
